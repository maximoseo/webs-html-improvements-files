## EXAMPLE 1: Brand Design System Extraction (from hue)

**Trigger:** "Extract the design system from dtapet.com"
**Skill:** html-redesign-mega-skill → Module B1

**Process:**
1. Load dtapet.com in browser
2. Extract: primary color (likely blue/green), background (#fff or brand color), typography (Hebrew-safe font)
3. Output CSS tokens file
4. Use those tokens in all subsequent redesigns

**Output example:**
```css
:root {
  --brand-primary: #2d6a4f;  /* extracted from dtapet.com */
  --brand-bg: #ffffff;
  --brand-surface: #f8f9fa;
  --brand-text: #212529;
  --brand-radius-md: 8px;
  --brand-font-heading: 'Heebo', Arial, sans-serif;
  --brand-font-body: 'Noto Sans Hebrew', Arial, sans-serif;
}
```

## EXAMPLE 5: AEO-Optimized Content Structure

**Trigger:** "Make this article AI-search ready"
**Skill:** html-redesign-mega-skill → Module B4

**Pattern:**
1. Add TL;DR box at top (direct answer)
2. Use semantic HTML: `<article>`, `<section>`, `<nav>`
3. Heading hierarchy: h1 → h2 → h3 (no skips)
4. FAQ as actual Q&A pairs
5. Data in `<table>` elements
6. Specific numbers: "43% of users" not "many users"
7. Named entities: "Ahrefs" not "SEO tools"
8. Code in `<pre><code class="language-*">`
