## UNIVERSAL RULES
1. WordPress cards: `<div>` + overlay `<a>` ‚Äî NEVER `<a>` as block container
2. Images: `loading=lazy` + `width` + `height` + `alt` always
3. CSS: prefixed classes (dta-), one `<style>` block
4. Responsive: 768px + 480px breakpoints
5. No inline JS ‚Äî CSS :hover only
6. Prompts: CTO-mode, MUST/NEVER/ALWAYS, no hedging
7. Design: no AI slop (no purple gradients, no pure black, no nested cards)
8. Models: OpenAI o3/o4-mini/GPT-4.1 + Anthropic Opus 4.5/Sonnet 4.5
9. GEO: direct answers first, named entities, specific stats
10. Preserve all knowledge ‚Äî add, never subtract
11. AEO: <8000 tokens/article, semantic HTML, llms.txt
12. RTL: dir=rtl + lang=he + Noto Sans Hebrew + line-height 1.8
13. Email HTML: tables not grid, inline styles, max 600px, absolute URLs
14. Accessibility: WCAG AA ‚Äî 4.5:1 body text, 3:1 large/UI, :focus-visible always
