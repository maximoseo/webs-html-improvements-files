### If Something Breaks in WordPress

| Symptom | Likely Cause | Fix |
|---|---|---|
| TOC numbers appear | Theme CSS overriding `list-style:none` | Add `!important` to `list-style:none !important;padding-left:0 !important` |
| Floating buttons hidden | Z-index conflict with theme | Increase `z-index` to `99999` |
| Scroll-to-top never appears | Script stripped by Classic Editor | Switch to Gutenberg Custom HTML block |
| Author section stacks vertically | Flex not supported or theme overriding | Add `flex-wrap:nowrap !important` or set fixed `width` on logo div |
| Accordion not animating | Style block stripped | Transitions are `@media` and `:hover` only — not critical for function |
| Images appear too large | Theme CSS overriding `max-width` | Images have `width:100%;max-width:100%` — should be fine in most themes |
| RTL direction appearing | Theme applying `dir="rtl"` | The `direction:ltr !important` inline styles override RTL in most cases |
