## Top 5 Most Valuable Finds

### 1. 🏆 `design-extract` (designlang) — *Highest Utility*
Extracts 8 design output files from any website including AI-optimized markdown, Tailwind config, shadcn CSS, Figma variables, W3C design tokens, responsive behavior across 4 breakpoints, and interaction states. This is a **design scraper and token generator** in one — feed it any URL and get a complete design system. Integrates as an agent skill with `--emit-agent-rules` generating CLAUDE.md fragments.

### 2. 🏆 `impeccable` — *Best Agent Skill for Design Quality*
18-command frontend design skill with the most comprehensive anti-pattern guidance found in any design skill. Has a curated list of banned "reflex fonts" (Inter, Fraunces, DM Sans...), 7 reference modules (typography, color, spatial, motion, interaction, responsive, UX writing), OKLCH color guidance, and a `craft` → `teach` → `polish` workflow. Fights LLM design monoculture systematically.

### 3. 🏆 `awesome-design-md` — *Largest Design Token Library*
69 ready-made DESIGN.md files for real products (Claude, Vercel, Stripe, Cursor, Raycast, etc.). The DESIGN.md format (inspired by Google Stitch) is becoming a standard for giving AI agents design context. This collection makes it trivial to mimic any established product's visual language.

### 4. 🏆 `figma-mcp-go` — *Best Figma Integration*
73 read/write tools via Figma plugin bridge — bypasses all REST API rate limits (Starter plan: 6 calls/month). Full design automation: create frames, set fills/strokes/auto-layout, clone components, manage prototypes, export assets. First MCP server with complete Figma read/write access at zero cost.

### 5. 🏆 `design-dna` — *Best Design Replication Workflow*
Three-phase workflow (Structure → Analyze → Generate) for extracting design identity as portable JSON across tokens, qualitative style, and visual effects (WebGL, particles, shaders). The DNA JSON can be version-controlled and shared — turns subjective "make it look like that" into a precise, reproducible specification.
