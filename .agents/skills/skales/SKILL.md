# SKILL: Skales - Local AI Desktop Agent (Windows/macOS/Linux)
**Source:** https://github.com/skalesapp/skales
**Domain:** code
**Trigger:** When deploying a local AI desktop agent, orchestrating multi-agent teams, or building a desktop app that integrates with Claude Code skills ecosystem

## Summary
One-click desktop AI agent (~300MB RAM) for Windows/macOS/Linux that supports 15+ AI providers, runs locally with Ollama, and imports Claude Code Agent Skills format. Features Codework (autonomous coding), Organization (multi-agent teams), Voice chat, and mobile pairing.

## Key Patterns
- Imports Claude Code Agent Skills format (SKILL.md) from GitHub URLs or local folders
- Organization: departments, specialized agents, team leaders, CEO auto-routing
- Codework: 3-panel GUI with live diffs for autonomous file editing
- Mobile pairing via QR code with end-to-end encrypted relay
- Migration importer from ChatGPT, Claude, OpenClaw, Hermes
- 139+ tools accessible from mobile after pairing

## Usage
Download from https://skales.app/download/latest-[platform]. Import skills via Settings > Skills > Add from GitHub URL. Build agent organizations via Organization tab.

## Code/Template
```
# Import community skills
Settings > Skills > Add > paste GitHub URL or local folder path
# Browse 1000+ community skills at: https://github.com/VoltAgent/awesome-agent-skills
```
