# SKILL: Minimalist Entrepreneur Skills — Sahil Lavingia's Framework
**Source:** https://github.com/slavingia/skills
**Domain:** marketing
**Trigger:** Use when validating startup ideas, finding first customers, setting prices, building sustainable growth, defining company values, or applying minimalist business principles to any product decision.

## Summary
10 Claude Code skills based on "The Minimalist Entrepreneur" by Sahil Lavingia (Gumroad founder). Covers the full founder journey: community → validate → MVP → processize → first customers → pricing → marketing → grow → culture → review.

## Key Patterns
- `/find-community` — find your community before building
- `/validate-idea` — test if a business idea is worth pursuing
- `/mvp` — scope your MVP, avoid over-building
- `/processize` — deliver value manually before writing code
- `/first-customers` — get to 100 customers one by one
- `/pricing` — set prices from day one
- `/marketing-plan` — content-driven scaling after PMF
- `/grow-sustainably` — profitable growth decisions
- `/minimalist-review` — gut-check any business decision

## Usage
Start with `/find-community` when exploring a new idea. Use `/validate-idea` before building. Use `/minimalist-review` before any major decision.

## Code/Template
```bash
# Claude Code
/plugin marketplace add slavingia/skills
/plugin install minimalist-entrepreneur
# Then use commands: /find-community, /validate-idea, /mvp, etc.
```
