# HTML PPT Studio

## Purpose
Browser-native presentation skill with an enormous design space: 36 themes × 15 templates × 31 layouts × 47 animations.

## Design Space
| Dimension | Count | Examples |
|---|---|---|
| Themes | 36 | Corp Blue, Midnight, Forest, Rose Gold, Arctic... |
| Templates | 15 | Business, Academic, Creative, Tech, Medical... |
| Layouts | 31 | Hero, Two-Col, Grid, Timeline, Quote, Full-Bleed... |
| Animations | 47 | Fade, Slide, Zoom, Parallax, Particle, Morph... |

## Key Features
- Zero dependencies — single `.html` file output
- Keyboard navigation + presenter mode
- Speaker notes (press S)
- Export: Print-to-PDF from browser
- RTL support
- Responsive for tablet/mobile view

## How To Apply
```
"Create an HTML presentation on: [topic], [slide count] slides, [theme], [animation style]"
"Make a product demo deck with the Midnight theme and particle animations"
"Build a 15-slide business presentation with corp-blue theme"
```

## Output
Single self-contained `presentation.html` file. Open in any browser. Share as file or host anywhere.

## Integration Notes
- Best for AI-generated decks meant to be sent as files
- Pairs with `marp-slides` — use Marp for Markdown-first editable decks, HTML PPT for visually rich browser decks
- Content works in RTL (Hebrew, Arabic) with `dir="rtl"` on body
