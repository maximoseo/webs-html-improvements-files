# SKILL: Founders Kit — Startup Resource Directory
**Source:** https://github.com/avinash201199/founders-kit
**Domain:** marketing
**Trigger:** Use when looking for curated startup tools, resources, or playbooks across product development, marketing, fundraising, design, analytics, or go-to-market strategy.

## Summary
A curated, community-maintained directory of essential startup resources: learning materials, tools/SaaS, funding resources, communities, design assets, analytics, startup programs (AWS/GCP credits), marketing channels, and a product submission directory.

## Key Patterns
- Categories: Learning, Courses, Podcasts, Fundraising, Website/Hosting, Design, AI Tools, Content/SEO, Automation, Payments, Email, CRM, Analytics, Marketing, User Feedback, Media/Video, Team Management
- Product Directory for listing your own startup
- Accepting PRs — actively maintained
- Submit your product via PRODUCT_SUBMISSION.md guide

## Usage
When a founder, PM, or developer needs to find the right tool for a specific startup stage. Browse categories or search by Ctrl+F. Especially useful for finding Content/SEO, Marketing, and AI tool resources.

## Code/Template
```
Key resource sections:
- Content & SEO → content marketing tools and SEO platforms
- Marketing Tools → acquisition channels and growth tools  
- AI Tools → AI-powered productivity and automation
- Startup Programs → AWS/GCP/Azure credits for startups
- Fundraising → pitch deck resources, angel networks
```
