# SkillClaw Evolver

## Purpose
Skills that evolve from every real interaction. Experience compounds across sessions, agents, devices, and users. Just talk — the skill system grows automatically.

## When To Use
- When building agent systems that should improve over time from usage
- When skills need to be shared and evolved across a team or user base
- When you want collective intelligence: one agent's learning benefits all
- When building production agent systems that adapt to real-world feedback

## How To Apply
1. Install: single command via skills marketplace
2. Interact naturally — each interaction is a learning opportunity
3. The system extracts patterns, updates skill definitions, propagates improvements
4. Skills are versioned: each evolution is traceable
5. Cross-device sync ensures all instances of the agent benefit

## Examples
- Agent repeatedly handles customer refunds → skill learns optimal refund flow patterns
- Multiple users correct the same response pattern → skill evolves the rule globally
- Agent discovers a better debugging sequence → pattern is distilled into a skill

## Integration Notes
- Broad compatibility: works across Claude Code, Cursor, Gemini CLI, Codex
- Skill evolution is agentic: the evolver itself is an AI agent
- Security: evolution proposals can require human approval before propagation
- Differentiator: vs static SKILL.md files, SkillClaw skills improve with usage
