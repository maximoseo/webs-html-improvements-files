# SKILL: Advertising Skills — Direct-Response Paid Media System
**Source:** https://github.com/realkimbarrett/advertising-skills
**Domain:** marketing
**Trigger:** Use when building paid advertising campaigns (Meta/TikTok/YouTube), writing direct-response copy, designing funnels, positioning offers, diagnosing ad performance, or building AI-driven marketing systems.

## Summary
A best-in-class skill library encoding Eugene Schwartz copywriting principles and paid media operator knowledge. Skills chain together for full-funnel campaigns: avatar extraction → offer → awareness mapping → mechanism → creative → funnel → conversion → diagnosis.

## Key Patterns
- Copy Chief layer: awareness levels, market sophistication, big idea/mechanism, headlines, leads, proof, objections, close
- Operator OS: scroll-stopping creative, ad angles, conversion paths, landing pages, email sequences, performance diagnosis, testing roadmap
- QA layer: generic-language-killer, claim-checker, consistency-auditor, final-polish
- Pre-built orchestrators: full-funnel, booked-call, campaign-diagnosis
- Foundation skills run first: avatar-extraction → voice-of-customer → offer-extraction → proof-harvester → positioning-core

## Usage
Run `avatar-extraction` and `offer-extraction` first before any creative. Chain through schwartz-awareness-mapper to determine which awareness level ad to run. Use performance-diagnosis for existing campaigns.

## Code/Template
```
# Full booked-call campaign flow:
avatar-extraction → offer-extraction → schwartz-awareness-mapper
→ mechanism-builder → ad-angle-multiplier → scroll-stopping-creative
→ conversion-path-builder → objection-crusher → generic-language-killer

# Install
npx skills add realkimbarrett/advertising-skills
```
