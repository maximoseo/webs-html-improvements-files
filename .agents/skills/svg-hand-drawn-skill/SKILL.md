# SVG Hand-Drawn Animation Skill

## Purpose
Generates animated SVG illustrations with a hand-drawn look — path draw-on animations, fill reveal effects, stroke morphing, and physics-like wobble.

## Output
- `preview.html` — standalone preview of the animation
- `player.js` — embeddable player for any website
- Inline `<svg>` with CSS/JS animation (no external dependencies)

## Animation Types
| Type | Effect |
|---|---|
| Path draw-on | SVG stroke animates as if being drawn |
| Fill reveal | Shape fills in from edge or center |
| Wobble | Physics-like bounce/settle |
| Morph | One shape transforms into another |
| Particle burst | Elements scatter and collect |
| Stagger | Sequential appearance of elements |

## How To Apply
```
"Create a hand-drawn SVG animation of: [subject]"
"Animate this SVG icon with a draw-on effect"
"Make a hand-drawn illustration of [concept] that animates on page load"
```

## Integration Notes
- Pure SVG + CSS + minimal JS — no external library required
- Works inline in any HTML page (WordPress-safe)
- Sizes: works at any scale (16px icon to fullscreen hero)
- `player.js` accepts options: `loop`, `autoplay`, `duration`, `easing`
- Pairs with `logo-generator-skill` for animated logo versions
