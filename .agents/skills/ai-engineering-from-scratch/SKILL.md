# SKILL: AI Engineering from Scratch - Comprehensive Course
**Source:** https://github.com/rohitg00/ai-engineering-from-scratch
**Domain:** code
**Trigger:** When learning AI engineering from fundamentals to autonomous agents, building reusable AI tools, or teaching AI concepts systematically

## Summary
260+ lessons across 20 phases (~290 hours) covering linear algebra through autonomous agent swarms. Every lesson produces a reusable artifact (prompts, skills, agents, MCP servers). Supports Python, TypeScript, Rust, and Julia. Uses Claude Code skills for AI-native learning.

## Key Patterns
- 20 phases: Math → ML → Deep Learning → Vision → NLP → Speech → Transformers → GenAI → RL → LLMs → LLM Eng → Multimodal → Tools → Agents → Autonomous → Swarms → Production → Ethics → Capstone
- Every lesson ships a reusable tool (prompt, skill, agent, or MCP server)
- AI-native learning: Claude Code skills test understanding per phase
- Built-in skills: /find-your-level, /check-understanding

## Usage
```bash
/find-your-level      # 10-question quiz to find starting phase
/check-understanding 3  # Per-phase 8-question quiz with feedback
ls phases/03-deep-learning-core/05-loss-functions/outputs/  # Reusable artifacts
```
