WORDPRESS CSS SURVIVAL NOTE
The display property is NOT on wp_kses_post() safe_style_css whitelist. Do not use display:flex, display:grid, or display:inline-flex in inline styles. Instead:
- For horizontal layouts: use float-based positioning, text-align:center with inline-block children, or percentage widths
- For accordion indicators: use float:left on the + span (not display:flex on summary)
- For product card grids: use display:inline-block with vertical-align:top (display:inline-block is pre-existing in WordPress themes and typically survives)
- For social links: use text-based inline links, not flex containers

Therefore:
