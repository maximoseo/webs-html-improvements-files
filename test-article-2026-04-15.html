Typography Hierarchy

Notice how headings use Playfair Display (serif, distinctive) while body text uses Source Sans Pro (sans-serif, readable). This creates visual interest without relying on color alone.

Subsection Example

Body text maintains 16-18px with 1.6 line-height for optimal readability. Captions or secondary text can drop to 14px with muted color (#6b7280).

Varied Spacing Rhythm

Section gaps vary: 80px, 96px, 64px. Never three consecutive identical spacings. This creates visual rhythm and prevents monotony. All spacing follows the 8px pattern (8, 16, 24, 32, 48, 64, 80, 96).

Frequently Asked Questions

Why use inline CSS instead of style blocks?

WordPress strips <style> blocks from post content. Inline CSS ensures your design survives WordPress rendering.

What's the one-H1 rule?

WordPress themes output the post title as H1. Your template should start headings at H2 to avoid duplicate H1s (bad for SEO).

Can I use this for RTL languages like Hebrew?

Yes! Add dir="rtl" to the <article> tag and change text-align to right. Margins/paddings will auto-mirror.
