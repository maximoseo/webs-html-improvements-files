## Mandatory output rules for future N8N generation
1. Hebrew RTL only.
2. Full WordPress Custom HTML block: `<!-- wp:html --> ... <!-- /wp:html -->`.
3. Inline styles on visible elements + scoped wrapper style block.
4. System fonts only: Arial, Helvetica, sans-serif.
5. No external CSS, JS, icon fonts, or Google Fonts.
6. Keep real Galoz links only: category, BEC product, optical encoder article, contact, WhatsApp.
7. Keep full article content and images. No skeleton placeholders.
8. No script tags in the final WordPress HTML. Schema can be handled separately by SEO plugin or workflow metadata.
9. Mobile-first responsive behavior via scoped media query.
10. CTA should ask the user to send shaft diameter, PPR, voltage, output type, controller, and environment.
