═══════════════════════════════════════════════════════
WORDPRESS WPAUTOP RESISTANCE (CRITICAL)
═══════════════════════════════════════════════════════

WordPress's wpautop filter converts newlines to <br> and wraps blocks in <p> tags.
This BREAKS icon+text alignment inside buttons. Follow these rules:

1. ALL buttons with SVG icons MUST have the SVG and text on the SAME LINE:
   CORRECT: <a ...><svg ...></svg> Button Text</a>
   WRONG:   <a ...>\n  <svg ...></svg>\n  Button Text\n</a>

2. NEVER put a newline between <svg> closing tag and the button text.
3. NEVER put a newline between the opening <a> tag and the <svg>.
4. Add vertical-align:middle to all SVG elements inside buttons.
5. Use display:inline-flex (not display:flex) on button links to prevent <p> wrapping.
6. Add flex-wrap:nowrap to flex button containers.
7. The floating back-to-top button goes on the RIGHT side (right:24px).
8. The floating contact button goes on the LEFT side (left:24px).
9. Both floating buttons must be on SEPARATE sides to avoid overlap.
