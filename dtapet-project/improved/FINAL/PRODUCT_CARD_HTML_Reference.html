3. בעיות נפוצות ותיקונים

בעיה: התמונה נחתכת או מעוותת

פתרון: השתמש ב-object-fit:contain במקום object-fit:cover

✅ נכון: style="width:100%;height:100%;object-fit:contain;padding:1rem;"

❌ לא נכון: style="width:100%;height:100%;object-fit:cover;"

בעיה: הכרטיסים לא באותו גובה

פתרון: הוסף min-height:380px לקלפ ו-margin-top:auto למחיר

זה מבטיח שכל הקלפים יהיו באותו גובה והמחיר תמיד בתחתית

בעיה: הגריד לא עובד בטלפון

פתרון: הוסף media query למסכים קטנים

@media (max-width: 768px) {
  .product-grid {
    grid-template-columns: 1fr;
  }
}

בעיה: WordPress מוסיף תגיות <p> ו-<br>

פתרון: הקפד למזער את ה-HTML (minify) ולהשתמש ב-shortcode או ב-Gutenberg blocks

או - השתמש ב-Classic Editor והדבק כ-HTML ישירות

✅ בדיקה: התבנית עובדת נכון

כל שלושת כרטיסי המוצרים למעלה מוצגים בצורה תקינה.

הם מכילים:

  • תמונה עם object-fit:contain (לא נחתכת)
  • כותרת מוצר
  • מחיר/קישור בתחתית
  • אפקט hover
